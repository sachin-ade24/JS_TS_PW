This file can be referred as TS installation guide.

   Steps:
      1. Download node and install it
      2. Check node version: node -v
      3. Check npm version: npm -v (Download NPM: npm install -g npm)
      4. Creat a package.json file: npm init -y --> creates package.json
      5. TypeScript compiler initialization: tsc --init
      6. Install ts-node (optional): npm install -g ts-node typescript --> Run file e.g.: ts-node yourFileName.ts 
      7. Install nodemon (optional): npm install -g nodemon --> Run file e.g.: nodemon yourFileName.ts 
      8. Install tsx i.e. typescript executor: npm install -g tsx --> Run file e.g.: tsx yourFileName.ts
         (tsx is optional but faster than ts-node and nodemon)
      9. Install yarn: npm install -g yarn (optional)
         then run yarn add -D typescript ts-node @types/node
         Run file e.g.: yarn ts-node yourFileName.ts
      10. Install code-runner (optional) from extensions (ctrl+shift+X)
      11. The file can be executed like this: node yourFileName.ts

Some mportant commands:
   1. Check byte code: node --print-bytecode yourFileName.js
   2. To check the TURBOFAN is started or not (check cold or hot code): node --trace-opt yourFileName.js
      (TURBOFAN will be started when the hot code is getting executed)
